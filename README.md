# 服科中心 H100 實習生技術測驗

這個專案包含兩個技術測驗題目的解答，展示了不同技術棧的應用。

## 專案結構

```
DAWDA/
├── 鍾孟儒_國立臺中科技大學_技術測驗(一)/    # 傷口評估系統
└── 鍾孟儒_國立臺中科技大學_技術測驗(二)/    # 個案管理追蹤看板
```

---

## 第一題：傷口評估系統

**技術棧：** Python

**功能說明：**
- 評估傷口狀態的醫療輔助系統
- 根據三項指標進行綜合判斷：
  - 傷口面積（cm²）
  - 疼痛等級（0-10）
  - 滲液量（None/Light/Moderate/Heavy）
- 自動分類傷口狀態（Critical / Fair / Good）
- 包含完整的輸入驗證機制

**執行方式：**
```bash
cd 鍾孟儒_國立臺中科技大學_技術測驗(一)
python main.py
```

**特色：**
- 內建 20 個測試案例自動驗證
- 支援互動式輸入模式
- 嚴謹的資料驗證邏輯

[詳細說明](./鍾孟儒_國立臺中科技大學_技術測驗(一)/README.md)

---

## 第二題：個案管理系統（CMS）追蹤看板

**技術棧：** HTML / CSS / JavaScript

**功能說明：**
- 個案訪視管理追蹤系統
- 視覺化呈現個案狀態
- 自動標記需追蹤個案（超過 7 天未訪視）
- 支援新增、編輯、刪除個案資料
- 分類顯示待追蹤與已完成個案

**執行方式：**
1. 使用 VS Code 開啟專案
2. 安裝 **Live Server** 插件
3. 開啟 `index.html` 檔案
4. 右鍵選擇「Open with Live Server」

**特色：**
- 響應式網頁設計
- 即時狀態更新
- 清晰的視覺化介面
- 模組化的 JavaScript 架構

[詳細說明](./鍾孟儒_國立臺中科技大學_技術測驗(二)/README.md)
